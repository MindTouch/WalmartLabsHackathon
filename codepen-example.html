This page includes three embedded MindTouch Touchpoints, served from walmartlabs.mindtouch.us -- a demonstration MindTouch-powered site filled with dummy content and provisioned for the August 12th, 2017 WalmartLabs / MindTouch Hackathon. Touchpoints allow customers to get product information at the point of sale, within the e-commerce experience.

Contextual Help Touchpoint

The walmartlabs.mindtouch.us link in the previous paragraph is actually wired to a Contextual Help Touchpoint and will open in a small contextual dialog, instead of a full page load. This link will also open in a contextual dialog but this link is not wired to the Touchpoint, so it will not -- it will instead open a full page experience in a new tab.

Search-in-Place Touchpoint

To the right of this paragraph is a Search-in-Place Touchpoint which returns product help articles based on the search queries provided. Search queries can be entered into the input field, and results will be returned from walmartlabs.mindtouch.us. Using a DOM event configuration process, it is configured to automatically search the walmartlabs.mindtouch.us site for articles that contain the phrase "content reuse". Results can be refined by drilling down into specific product categories.

Contextual Help Button Touchpoint

This button is configured to open a walmartlabs.mindtouch.us URL, like the links above, however the dialog wired to this button is configured slightly differently. Some buttons have been removed and there is a different search input hint. Touchpoints can be configured in many different ways, and even styled with CSS and LESS!
